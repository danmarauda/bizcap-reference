## Cloned repositories

All paths are relative to `bizcap-reference/repos`.

### 1. `heroui-inc/heroui`
- Path: `repos/heroui`
- Description: Beautiful, fast and modern React UI library (formerly NextUI).
- Use for:
  - Component APIs and best practices.
  - Theming and Tailwind integration patterns.
  - Example layouts and accessibility patterns.

### 2. `heroui-inc/heroui-native`
- Path: `repos/heroui-native`
- Description: HeroUI for React Native.
- Use for:
  - Mobile UI components to inspire the Expo app.
  - Patterns for theming across platforms.

### 3. `heroui-inc/heroui-cli`
- Path: `repos/heroui-cli`
- Description: CLI for integrating HeroUI into projects.
- Use for:
  - Understanding how the CLI wires up Tailwind and config files.
  - Potentially borrowing patterns for automating setup in our own stack.

### 4. `heroui-inc/next-app-template`
- Path: `repos/next-app-template`
- Description: Next.js app directory template with HeroUI v2 and Tailwind v4.
- Use for:
  - Reference implementation for Next.js 15 + App Router + HeroUI.
  - Tailwind v4 config and layout patterns.
  - Ideas for integrating HeroUI into our `apps/web`.

### 5. `SteveSuv/remix-words-funny`
- Path: `repos/remix-words-funny`
- Description: Fullstack app using Remix + tRPC + Drizzle + PostgreSQL + HeroUI + Jotai + Docker.
- Use for:
  - Real-world usage of HeroUI in a production-style app.
  - Form handling, layout, and state management patterns.

### 6. `heroui-inc/heroui-native-example`
- Path: `repos/heroui-native-example`
- Description: Demo project for HeroUI Native.
- Use for:
  - End-to-end example of a mobile UI built with HeroUI Native.
  - Navigation and screen composition.
